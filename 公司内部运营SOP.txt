一、日常运营 SOP（每日 / 每周）
1.1 数据监控（每日 9:00-10:00）
核心指标：GMV、流量、点击率、转化率、客单价、退款率、DSR
平台：淘宝 / 天猫、京东、拼多多、抖音小店
异常处理：指标波动 > 10% 立即排查（流量 / 转化 / 价格 / 库存 / 评价）
输出：《每日运营数据简报》
1.2 商品管理（每日 10:00-12:00）
上架：类目 / 属性 / 标题 / 主图 / 详情页 / 价格 / SKU 校验
库存：每日 16:00 核对，低于安全库存（如 50 件）触发预警，协调补货 / 下架
优化：每周分析低效品（访客 < 50 / 日、转化率 < 1%），优化标题 / 主图 / 价格
上下架：按平台流量高峰（10:00-12:00、19:00-23:00）定时上架
1.3 营销推广（每日 13:00-16:00）
付费推广：直通车 / 引力魔方 / 京准通 / 多多搜索，关停低 ROI 计划，加投高 ROI
活动报名：筛选平台活动（秒杀 / 满减 / 百亿补贴），提交素材与价格
内容运营：短视频 / 直播 / 图文发布，监控播放 / 互动 / 转化
1.4 订单与售后（实时监控）
订单审核：地址 / 优惠 / 库存校验，异常订单 2 小时内处理
发货：48 小时内出库，物流单号同步
售后：差评 24 小时内响应，高频问题更新 FAQ，客诉升级专人处理
1.5 每日复盘（17:30-18:00）
目标完成率、亮点 / 问题、明日优先级 3 件事
异常记录：违规 / 差评 / 物流 / 库存，同步至团队
二、大促运营 SOP（618 / 双 11 / 周年庆）
2.1 筹备期（活动前 15-30 天）
目标：GMV、新客、转化率、ROI
方案：玩法（满减 / 优惠券 / 秒杀 / 拼团）、价格体系、库存规划
素材：主图 / 详情页 / 海报 / 短视频 / 直播脚本
培训：客服 / 主播 / 运营熟悉规则与话术
测试：优惠券核销、库存扣减、支付流程
2.2 预热期（活动前 7 天）
推广：站内（首页 / 弹窗 / 购物车）、站外（社群 / 小红书 / 抖音）
蓄水：加购 / 收藏 / 预售，短信 / PUSH 触达
监控：预热数据、竞品动态、价格合规
2.3 爆发期（活动当天）
实时监控：流量 / 转化 / GMV / 库存 / 客诉
动态调整：推广预算、价格、库存、客服人力
氛围营造：战报播报、直播间互动、社群促单
2.4 返场期（活动后 1-3 天）
未履约：加急发货、未用券引导复购
数据快报：核心指标汇总，同步管理层
2.5 复盘期（活动后 5-7 天）
数据深度分析：目标达成、渠道 ROI、用户画像
问题诊断：流量 / 转化 / 库存 / 售后 / 合规
输出：《大促复盘报告》+ 优化 Action
三、合规风控 SOP（红线禁止）
3.1 价格合规
禁止虚假标价、虚构原价、违规降价
大促价格需提前审核，符合平台与公司政策
禁止价格欺诈、诱导消费
3.2 营销合规
优惠券 / 满减规则清晰，禁止冲突与误导
禁止夸大宣传、绝对化用语（最 / 第一 / 顶级）
素材版权合规，禁止盗用
3.3 数据合规
禁止刷单、刷评价、虚假交易
用户数据保护，符合隐私法规
